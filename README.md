# Backend Python - Movie List API

Projeto desenvolvido como parte de um desafio técnico para Backend Python.

## Objetivo

Desenvolver uma API para manipulação e consulta de dados de filmes utilizando Python e boas práticas de desenvolvimento.

## Tecnologias

- Python 3.x
- Flask/FastAPI
- Pandas
- Pytest

## Estrutura

```
app/
tests/
data/
docs/
```

## Instalação

```bash
pip install -r requirements.txt
```

## Execução

```bash
python app.py
```

## Testes

```bash
pytest
```
